# Boxstov

Zombies survival co-op game using Phaser and WebRTC.

## How To Use

### Requirements

This project requires `Node 18`

### Client

```bash
$ cd client
# Install dependencies
$ npm install
# Start the local development server
$ npm run start
```

### Server

```bash
$ cd server
# Install dependencies
$ npm install
# Start the local development server
$ npm run dev
```
